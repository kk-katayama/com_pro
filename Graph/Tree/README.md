# 木構造 <br>

辺が頂点数-1で連結のグラフを木というよ。

## 木の直径<br>

一番遠い葉同士の距離を木の直径という。d[v][w] をv->w のパスの重みの総和として、木の直径 = max{d[v][w]}。<br>
