# 拡張ユークリッドの互除法  

蟻本の序盤（二分探索より前！）に登場する謎のアルゴリズム。使い所がいまいち不明だったけど、最近(2020/09/28)使うようになってきた。  

## なにそれ
整数a, bが与えられたとき、ax + by = gcd(a, b) を満たす整数(x, y)は必ず存在して、ユークリッドの互除法を拡張することで求めることができる。
